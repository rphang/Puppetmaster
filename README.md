# Puppetmaster

Puppetmaster is a tool for retrieving and managing the time schedules of students in Master 2 at the University of Lyon 1.

## Installation

### Requirements

- Node.js
- npm
- MongoDB

### How-to

#### From repo

1. Clone the repository

```bash
git clone ..
```

2. Install dependencies

```bash
npm install
```

3. Run the server

```bash
npm start
```
